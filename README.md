A fast-paced 2D survival game where the player must defend themselves against increasingly aggressive waves of enemies in a village environment. The goal is to survive as long as possible while managing health, positioning, and enemy pressure.

The player can attack enemies, retreat into houses for temporary healing, and strategically navigate terrain to avoid being overwhelmed. As time progresses, enemy waves grow stronger, creating a dynamic difficulty curve that tests survival skills and decision-making



🎮 Controls
⬆️⬇️⬅️➡️ — Move player
Space — Attack
E — Enter / Exit house
Left Shift:Increase speed
